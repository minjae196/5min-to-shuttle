# 5min-to-shuttle

셔틀버스 시뮬레이션 프로젝트입니다. 이 프로젝트는 특정 조건과 시뮬레이션 환경에서 셔틀버스 운행을 시뮬레이션하여 결과를 분석하는 데 목적을 두고 있습니다.

## 🛠 프로젝트 특징

- 셔틀버스 경로와 정차 시간을 시뮬레이션
- 승객의 승하차 분석

## 🚀 주요 기능

- **시뮬레이션 설정**: 셔틀버스의 운행 시간, 경로, 정류장 수를 설정
- **데이터 분석**: 시뮬레이션 결과를 기반으로 이동 경로 최적화
- **사용자 정의 가능성**: 다양한 인풋 조건과 확률 분포를 사용하여 실험적인 시뮬레이션 실행

## 🔧 설치 방법

1. **필요한 소프트웨어 설치**
   - Python 3.8 이상이 필요합니다.
   - 필수 라이브러리를 설치합니다:
     ```bash
     pip install -r requirements.txt
     ```

2. **프로젝트 클론**
   ```bash
   git clone https://github.com/minjae196/5min-to-shuttle.git
   cd 5min-to-shuttle
   ```

3. **실행**
   - 기본 실행:
     ```bash
     python main.py
     ```
